\section{Example Script}
The provided example script demonstrates some of the functionality of the
pre-existing code. The first few lines initialize the project, and then there is
an example of some basic polynomil arithmetic:
\begin{Verbatim}
    x = x_poly()
    p1 = 2x^3 + 4x^2 - 3x
    p2 = 2x^4 - 4x^2 - 3x + 3

    @show p1*p2
    @show p1^3
\end{Verbatim}
Output:
\begin{Verbatim}
    p1 * p2 = 4x⁷ + 8x⁶ - 14x⁵ - 22x⁴ + 6x³ + 21x² - 9x
    p1 ^ 3 = 8x⁹ + 48x⁸ + 60x⁷ - 80x⁶ - 90x⁵ + 108x⁴ - 27x³
\end{Verbatim}
